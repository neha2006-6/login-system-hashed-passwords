# Login System with Hashed Passwords 🔐

A secure login system built with Python that stores 
passwords safely using hashing.

## Features
-  User Registration
-  Secure Password Hashing
-  User Login
-  Data stored in JSON file

## How it Works
1. User registers with username and password
2. Password is hashed using SHA256
3. Hashed password is stored safely
4. Login compares hashed passwords

## How to Run
```bash
python login_system_v2.py
```

## Technologies Used
- Python
- hashlib (for hashing)
- JSON (for storage)
